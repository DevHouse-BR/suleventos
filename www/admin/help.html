Pagina Inicial
Senha do Administrador = Os campos "Nova Senha" e "Confirmação" devem ambos ser preenchidos com a nova senha do administrador. Caso as duas não coincidirem o programa não executará a troca. A senha de administrador é único meio de acesso a administração de conteúdo do site.
Quantidade de destaques = Determina a quantidade de eventos recentes vao aparecer no quadro a direita do site.
 
Enquetes
Nova Enquete = Cria uma nova pesquisa.
Gerenciar Enquetes = Apaga ou altera enquetes antigas.
Tamanho da enquete = Este campo altera a altura da janela da enquete.
É útil regular a altura após a criação de uma nova enquete para que o texto da mesma não ultrapasse a área reservada.
A última enquete criada seguindo a ordem de data é a enquete que aparece nas páginas. Mas só aparece se estiver "ativa".
 
Editor de HTML online
Com este editor é possivel adicionar efeitos às páginas sem conhecer a linguagem HTML. O editor é semelhante aos editores de texto da microsoft como o Word e o Wordpad. Um dos recursos do editor é a inserção de imagens junto ao texto digitado. Ao clicar no ícone de imagens o editor chama um software de gerenciamento das imagens possiveis de serem inseridas em textos. Estas diferem das fotografias dos eventos pois o site as trata de forma diferente. Com o gerenciador de imagens é possivel adicionar novas imagens ou até mesmo apagá-las (tomar cuidado pois ao apagar uma imagem que está sendo usada em um texto faz com que a imagem não apareça mais nele. Para inserir uma imagem no texto basta selecionar se deseja inserir o thumbnail da imagem ou a imagem em sí. A diferença é que inserindo o thumbnail a imagem aparece reduzida no texto mas torna-se um link para a imagem em tamanho natural. A opção de alinhamento determina como a imagem vai aparecer no entre o texto a direita, centralizado ou a esquerda. São aceitas somente imagens do tipo "jpg" ou "gif".
 
Eventos
Tipo = É possível adicionar novos tipos de eventos como Bodas, Aniversário, Debutante, etc. Não é possivel renomear nem apagar os tipos por isso deve-se tomar bastante cuidado ao criá-los.
Nomes = Os nomes da(s) pessoa(s) envolvidas no evento como o nome dos noivos, da debutante entre outros.
Data = data em que aconteceu o evento. O formato da data deve seguir o padrão dd/mm/aaaa. Pode-se utilizar o calendário para escolher a data.
Local = Local onde foi celebrada a cerimônia ou a festa. Descrição = Breve descrição do acontecimento. No máximo de 255 caracteres.
Requer senha = Caso este evento não for aberto a visitação publica então informar Requer Senha = sim e preencher o campo "email" com o email da pessoa que organizou o evento para que ela receba a senha e um link para o site.
Fotografia de Destaque = Porcurar no computador local a foto a ser enviada para o site para servir de destaque para o evento. Deve-se escolher a melhor foto ou a foto dos noivos.
Adiciona Foto = Adicionar mais fotografias para este evento.
Adicionar Parceiros = Para associar parceiros já cadastrados ao evento selecione na caixa "Parceiro" o nome do parceiro e clique em adicionar. Repita o procedimento até que todos os parceiros que trabalharam neste evento estejam associados ao mesmo. Caso seja adicionado um parceiro errado é possivel remover o parceiro do evento sem apagá-lo do sistema.
Apagar Foto = Apaga a foto do evento e do servidor das paginas.
Alterar foto de destaque = Sobre-escreve a foto anterior por uma nova.
Adiciona Foto = Adicionar mais fotografias para este evento.
Editar parceiros = Altera os parceiros que participaram do evento.
 
Parceiros
Imagem = Fotografia destacando o trabalho do parceiro, seu logotipo ou o seu cartão.
Nome = Nome do parceiro.
Tipo do Parceiro = É possível adicionar novos tipos de parceiros como Aluguel de Trajes, Automóveis Antigos, Arranjos, Floricultura, etc. Não é possivel renomear nem apagar os tipos por isso deve-se tomar bastante cuidado ao criá-los.
Descrição = Descrever o serviços prestados pelo parceiro. Maximo de 255 caracteres.
 
Fale Conosco
É possivel informar o email para receber as mensagens enviadas através do site.
